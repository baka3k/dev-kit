# Markdown Body Templates

Adapt these outlines to the task; omit sections that add no information.

## Plan

```markdown
---
title: "{Plan title}"
status: pending
created: YYYY-MM-DD
---
# {Plan title}
## Overview
## Phases
## Dependencies
## Success Criteria
```

## Phase

```markdown
# Phase {NN}: {Name}
## Context
## Requirements
## Architecture
## Related Files
## Implementation Steps
## Todo
## Risks
## Success Criteria
```

## Report

```markdown
---
type: {report type}
date: YYYY-MM-DD
---
# {Report type}: {Subject}
## Summary
## Findings
## Recommendations
## Unresolved Questions
```

## Log

```markdown
---
date: YYYY-MM-DD
topic: {topic}
---
# Log: {Topic}

## Decisions
## Reflection
## Next Steps
```

## Document

```markdown
# {Title}

```

## Architecture Decision

```markdown
# ADR-{NNN}: {Decision}
- **Status:** proposed | accepted | deprecated | superseded
- **Date:** YYYY-MM-DD
## Context
## Decision
## Consequences
## Alternatives Considered
```
