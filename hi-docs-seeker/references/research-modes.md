# Research Modes

Choose the narrowest mode that answers the request.

| Mode | Primary evidence | Verification |
| --- | --- | --- |
| Topic | Exact official guide or API page | Versioned reference or official example |
| Version | Versioned docs and release notes | Package registry or official tag |
| Migration | Migration guide and breaking-change notes | Old and new version references |
| Compatibility | Official support matrix or requirements | Release notes for each boundary version |
| Overview | Official introduction, concepts, and API index | Current release page |
| Error or bug | Official troubleshooting page or issue tracker | Fix release, changelog, or maintainer response |

For one focused question, research directly. Split into independent read-only tracks only when parallel work is explicitly allowed and the request spans multiple products, versions, or source types. Keep scopes distinct and deduplicate evidence during synthesis.
